This folder needs the file: autounattend.xml

ONCE, before uploading to GitHub (or before first using the stick):

1. Open this link in your browser:
   https://raw.githubusercontent.com/memstechtips/UnattendedWinstall/main/autounattend.xml
2. Right-click -> "Save as" -> save it as autounattend.xml RIGHT HERE
3. Commit it to the GitHub repo (MIT license, see LICENSE-UnattendedWinstall.md)

Alternatively, "create-stick.bat" downloads it automatically while filling
the stick - but then end users won't get the file via the repo ZIP.

This file (README-FIRST.txt) can be deleted afterwards.
